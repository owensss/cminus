cminus
======

coursework

Desc
----------

> a cminus compiler running on spim
a cminus IDE build by Qt5

Deploy
------

### ToolChain
*  bison, flex
*  g++, support c++11 with -std=c++0x enabled
*  gnu make
*  Qt5

### Make
*  goto cminus/, run make
*  build UI/UI.pro
*  cp cminus/cminus UI-build/

### Usage
* cminus filename
